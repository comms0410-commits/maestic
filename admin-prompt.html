💬 LINE · Claw Webhook 설정 연동 설정

Claw 에이전트 또는 LINE Messaging API webhook URL을 등록하면, 상태 변경 시 자동으로 알림이 발송됩니다.

Claw 에이전트의 Incoming Webhook URL
📖 Claw 연동 가이드
1. Claw 에이전트 → Webhook 설정 → Incoming URL 복사
2. 위 "Claw Webhook URL" 칸에 붙여넣기
3. "Webhook 알림 활성화" 체크
4. 저장 후 테스트 발송으로 확인

💡 발송되는 이벤트: 기사 Pick 제출, 대본 검토 요청, 수정 의견, 월별 현황 업데이트, 승인 완료
